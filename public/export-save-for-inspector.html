Export game save for inspector

Use this page while a solo game is running on play.html (same browser tab session). The inspector runs on your machine: npm run inspect:save-economics -- --file=…

Option A — In-game download (recommended)

  1. Open Save / Load from the game menu.
  2. Click Download save file — saves airwave-empire-YYYY-MMDD.json.
  3. Run: npm run inspect:save-economics -- --file=/path/to/that-file.json

Option B — Console export (this tab)

With the game loaded in this window, click below or paste the snippet in DevTools console.

(function () {
  if (typeof G === 'undefined' || !G || !G.stations) {
    console.warn('No active G — load a solo game first.');
    return;
  }
  var label = (G.year || '?') + ' P' + (G.period || '?') + ' export';
  var payload = typeof saveGame === 'function'
    ? saveGame(label)
    : { v: 1, saved: new Date().toISOString(), label: label, G: G };
  var blob = new Blob([JSON.stringify(payload, null, 2)], { type: 'application/json' });
  var a = document.createElement('a');
  a.href = URL.createObjectURL(blob);
  a.download = 'airwave-empire-export-' + (G.year || 'game') + '.json';
  a.click();
  URL.revokeObjectURL(a.href);
  console.log('Exported', payload.label, 'stations', G.stations.length);
})();

Option C — Autosave from localStorage

Keys: airwave_empire_autosave (current) or legacy wavelength_autosave.

  1. DevTools → Application → Local Storage → your play origin.
  2. Copy the value into autosave.json (valid JSON).
  3. Run the inspector on that file.

Inspector flags

npm run inspect:save-economics -- --file=save.json
npm run inspect:save-economics -- --file=save.json --station=KXXX
npm run inspect:save-economics -- --file=save.json --market=sanfrancisco --year=1980 --period=1

Output: tmp/save_economics_inspection.json and .md
